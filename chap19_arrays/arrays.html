How to recognize an arrat

  • Array.isArray(array_name); // return true or false
    ECMAScript 5이하의 오래된 브라우저에서는 지원하지 않음
  • 나만의 isArray() 함수를 만들기
    function isArray(x) {
    return x.constructor.toString().indexOf("Array") > -1;
    }
    object 프로토타입에 'Array'라는 단어가 있으면 true를 반환
  • instanceof 는 주어진 생성자가 객체를 생성하면 true를 반환
    array_name instanceof Array;
